# Impeccable Design Context

## Design Context

### Users
Personal power user managing and browsing a large Telegram media archive. Self-hosted, solo use. The user is technical, expects keyboard shortcuts, fast filtering, and efficient navigation through thousands of media items. They value density of information without visual clutter.

### Brand Personality
**Minimal, calm, refined.** The interface should feel like a quiet, well-organized space — not shouty or attention-seeking. Every element earns its place. The design conveys competence through restraint.

### Aesthetic Direction
- **Reference**: Apple Photos — polished typography, elegant transitions, refined grid layouts, generous but purposeful whitespace
- **Anti-reference**: Cluttered dashboards, Discord-style busy UIs, generic Bootstrap admin panels
- **Theme**: Dark-first with a refined light mode. The current teal-tinged dark palette is a strong foundation
- **Typography**: Manrope provides clean, modern readability — lean into weight contrast (400 for body, 600-700 for emphasis) rather than size contrast

### Design Principles

1. **Quiet confidence** — The UI should feel assured and understated. No gratuitous decoration, no competing focal points. Let the media be the star.

2. **Purposeful density** — A power user wants information-rich views, but every pixel of density must be intentional. Tight spacing with clear hierarchy, not cramped chaos.

3. **Smooth and responsive** — Transitions should feel physical and natural (Apple-style easing). Interactions should acknowledge input instantly. No jank, no layout shifts.

4. **Keyboard-first, touch-aware** — Design for keyboard power users first, but ensure touch interactions feel equally considered. No hover-only affordances.

5. **Consistency over novelty** — Use the same patterns repeatedly. Same spacing, same border radii, same animation curves. Predictability builds trust and speed.

### Color Foundation
- **Accent**: Sky blue (#0284c7) — trustworthy, calm, distinctive against teal-dark surfaces
- **Semantic**: Emerald success (#10b981), red danger (#dc2626), amber warning (#d97706)
- **Surfaces**: Teal-tinged darks (current palette) — distinctive without being heavy

### Typography Scale
- **Font**: Manrope (sans-serif)
- **Weights**: 400 (body), 500 (labels/UI), 600 (emphasis), 700 (headings)
- **Approach**: Size hierarchy should be subtle — rely on weight and opacity more than large size jumps

### Motion
- **Easing**: ease-out for entrances, ease-in for exits, ease-in-out for state changes
- **Duration**: 150-200ms for micro-interactions, 250-350ms for layout transitions
- **Philosophy**: Motion should clarify spatial relationships and confirm actions, never delay the user
