# Backend Agent Rules

## Package Manager

- Use `uv` as the package manager and runner
- Run with `uv run fastapi dev` for development
- Add dependencies with `uv add <package>`

## Code Style

- Use `ruff` for linting and formatting
- Python 3.12+

## Stack

- **Framework**: FastAPI
- **Runtime**: Python 3.12
- **Package Manager**: uv

## Conventions

- Use async route handlers
- Type-annotate all function signatures
